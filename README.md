# Lojita-web
Este projeto foi desenvolvido do Figma ao React utilizando o conceito de Design System. A aplicação trata-se de um marketplace para administrador, vendedores e compradores. Gerencie usuários, chats, pedidos, produtos, favoritos e carrinho de forma prática e rápida.

# Tecnologias
- Vite
- React
- React Hook Form
- React Query
- React Router DOM
- TypeScript
- Zod
- Axios
- HeroUI
- Tailwindcss
- Lucide Icons
- ESLint + Prettier

# Desafios
- Implementar componentes escaláveis com HeroUI
- Construir uma aplicação web com renderizações condicionais otimizadas e seguras para cada tipo de usuário
- Implementar interfaces otimizadas, responsivas e de boa usabilidade

# Observações
- Alguns dados como usuários, produtos e pedidos cadastrados podem sumir já que o site online é uma versão com dados mockados
